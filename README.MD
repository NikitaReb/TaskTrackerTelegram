<h1 align="left">Подсистема "Планировщик задач Телеграм"</h1>

Подсистема родилась из необходимости оперативного сбора ошибок от пользователей, без флуда в общем диалоге.
Подсистема представлена в виде расширения, которое работает совместно с подсистемой БСП "Регламентные задания".

<h3 align="left">Принцип работы:</h3>

  - Системный администратор создает бота, делает предварительные настройки системы.
  - Передает ссылку на боту пользователям. Пользователи при наличии вопросов отправляют сообщение.
  - На каждое сообщение создается документ "Задача пользователя Телеграм" создается".
  - При смене статуса или ответа в документе, отправляется сообщение пользователю в Телеграм.


<h3 align="left">Настройки подсистемы:</h3>

  - Создать бота в ТГ https://telegram.me/BotFather. Скопировать токен.
  - Создать элемент справочника "Токены управления Телеграм-ботами" и подставить токен в наименование.
  - Открыть форму настроек "Настройки планировщика задач Телеграм" и выбрать токен в качестве основного.
  - Установить расписания регламентным заданиям:
    "Загрузка задач пользователей из Телеграм" и "Отправка ответов по задачам пользователей Телеграм".
  - Ожидать получения сообщений в документе "Задача пользователя Телеграм".
